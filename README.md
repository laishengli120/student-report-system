# 班主任小助手：期末通知书 AI 生成与送达确认系统

## 简介
面向农村小学期末通知书发放、评语撰写、假期安全告知和家校送达确认的低成本智能信息系统。系统原型和主要代码由一线班主任借助 DeepSeek 多轮提示词生成、调试和整合，支持上传 Excel 学生信息、AI 生成可审核评语、教师审核发布、家长手机端查看通知书并手写签名确认。

本项目不把 AI 作为替代教师评价学生的工具，而是把 AI 初稿、教师把关、电子通知送达、家长签收留痕连成闭环，重点解决乡村学校期末事务集中、纸质通知送达慢、留守儿童父母难以及时知情、回执统计负担重等真实问题。项目适合经费有限、仍以纸质通知和微信群沟通为主的学校参考复现。

## 功能特性
- **学生信息管理**：上传 Excel 表格，自动导入学生姓名、职位及语文、数学、英语、科学、道德与法治等字段。
- **AI 评语生成**：内置 12 种成长画像标签（优秀全能型、乖巧可爱型、勤奋自觉型等），AI 根据标签生成 60-90 字评语初稿，不直接按成绩评价学生。
- **教师审核工作流**：上传后可进入草稿状态，支持逐条或批量生成 AI 评语；教师可修改、保存、确认后再统一发布。
- **分数等级规则**：后台可配置“最低分=等级”的映射规则，并设置家长端仅显示等级、仅显示分数或同时显示分数与等级。
- **假期安全告知书**：提供寒假版、暑假版、通用版模板，支持教师编辑、预览和调用 AI 优化文案。
- **低成本送达确认**：家长通过手机浏览器查看电子通知书，父母在外务工也可及时知情并手写签名确认。
- **签收看板**：后台统计已签收、未签收人数，可筛选查看学生签收状态和签署时间，便于班主任跟进。
- **时间参数配置**：后台可自定义放假、下学期报到及正式开学时间，通知书自动生成对应寒假/暑假表述。
- **双端适配界面**：教师端支持桌面侧边栏和移动端底部导航，家长端适配手机阅读和签名。
- **管理员认证与 AI 接口设置**：基于 session 的管理员登录；支持 DeepSeek 及其他 OpenAI SDK 兼容接口。

## 技术栈
- Python 3 / Flask
- SQLite（数据存储）
- Pandas（Excel 解析）
- OpenAI SDK（调用 DeepSeek 等兼容接口生成 AI 评语）
- Jinja2 + Tailwind CSS（前端页面渲染）
- DeepSeek（辅助需求拆解、代码原型生成、提示词优化与系统调试）

## 快速开始
```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py
```

默认访问地址：`http://127.0.0.1:5000`

- 家长查询页：`/parent`
- 管理后台：`/admin/login`（默认账号 admin / admin123）

## AI 评语配置
在管理后台设置 AI 接口参数：
- API Key：你的 API 密钥
- Base URL：默认 `https://api.deepseek.com`
- Model：默认 `deepseek-v4-flash`

提示词已内置 12 种学生类型标签，AI 会聚焦于学生身心发展、性格品行和习惯养成，不涉及成绩评价。

## 负责任使用说明
- AI 评语仅作为初稿，必须由教师审核、修改、确认后发布。
- 演示、参赛和分享时请使用匿名数据，不出现学生正面画面、真实联系方式、家庭住址和敏感家庭信息。
- 如果材料中出现 AI 生成内容，应明确标注“AI生成初稿，经教师审核后使用”。
- 正式部署时建议修改默认管理员密码，并按学校要求补充家长身份校验或学生编号查询。
